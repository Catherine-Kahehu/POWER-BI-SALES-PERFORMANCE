# POWER-BI-SALES-PERFORMANCE
## Objective
The objectives of this project are; To clean and transform raw sales data using Power Query, to create calculated measures using DAX, to analyze sales performance trends, to build interactive dashboards and reports and to provide meaningful insights for business decision-making.
## Dataset used
Superstore Dataset
## Technologies used
Power BI Desktop, Power Query, DAX, Excel Dataset
## Describe the technologies used
Power BI was used to create interactive dashboards and visual reports, Power Query was used for extracting, cleaning and transforming raw data, DAX was used to create calculated and performance measures while Excel was used as the source data file.
## Possible Usage & Application of the data
The Superstore dataset can be used in businesses to support decision making, improve sales performance, analyze customer behavior, evaluate product performance, trend analysis and development of business intelligence dashboards.
## Results/Key Findings
Based on the analysis of the Superstore dataset and dashboard insights, key findings were that certain product categories contributed more revenue, high sales did not always lead to high profit, sales trends changed over time while sales and profit varied across regions
